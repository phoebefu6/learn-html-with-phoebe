# learn html with phoebe

Six 45-minute sessions + three self-paced deep dives taking a total beginner from their very first tag to their own page live on the internet.

**The twist:** every example in the course is an editable playground - real HTML source in a code panel next to a sandboxed live preview. Type, break, fix, watch. No installs, no setup, learning happens in the page itself.

**The spine:** one running project. "Your corner of the internet" - a personal page that grows every session (structure → links → images and semantic layout → tables and a contact form → a CSS makeover) and ships to GitHub Pages on a real URL in the Session 6 finale.

## Sessions

| # | Session | Difficulty | Covers |
|---|---------|-----------|--------|
| 1 | Your first page, in a plain text file | 🟢 | elements, attributes, the document skeleton, headings |
| 2 | Words that link | 🟢 | meaningful text, lists, links, anchors, the head + favicon |
| 3 | Pictures and structure | 🟡 | images + alt discipline, video/audio, semantic layout |
| 4 | Tables and forms | 🟡 | accessible data tables, forms, labels, validation |
| 5 | Enough CSS to be proud | 🟠 | selectors, color + type, box model, flexbox starter |
| 6 | Polish and ship | 🔴 | validator, a11y audit, viewport, GitHub Pages - LIVE |
| 6.1-6.3 | Deep dives | self-paced | no-JS interactivity, SVG + embeds, SEO + social cards |

Each session: 45 minutes live (3 welcome / ~20 concepts / ~17 build-along / 5 Q&A), full self-study depth on the same page, homework, a 3-question quiz, honest source-coverage notes, and a cheat sheet.

Covers the working content of MDN Learn's "Structuring content with HTML" path (25 lessons), a starter subset of MDN CSS basics, and web.dev's Learn HTML course (22 chapters) - overlap taught once, deltas in the deep dives. MDN's own challenges and web.dev's quizzes stay with the originals.
